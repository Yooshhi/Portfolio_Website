JavaScript Showcase: Adding Interactivity

Introduction

JavaScript makes web pages interactive. Unlike HTML (structure) and CSS (design), JavaScript allows users to interact with content, update data dynamically, and enhance the user experience.

Interactive Example: Add Items to a List

This demo shows how JavaScript can take user input and add it directly to the page without reloading.

How it Works

  1. You type something into the input box.
  2. When you click "Add", a JavaScript function runs.
  3. The function creates a new list item and inserts it into the existing list.
  4. You can also remove the last item or highlight all items using the new buttons.
