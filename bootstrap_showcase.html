Web Development Basics

This page introduces the core building blocks of web development using Bootstrap for styling.

Introduction

Web development is the process of building and maintaining websites. It has three main pillars:

  1. HTML - the structure of the webpage.
  2. CSS - the design and styling of the webpage.
  3. JavaScript - the interactivity and functionality of the webpage.

HTML Basics

HTML provides the structure of a website through different elements and tags. Here are some examples:

Tag Purpose
<h1> - <h6> Defines headings from largest to smallest.
<p> Defines a paragraph of text.
<a> Creates a hyperlink.
<img> Embeds an image.
<table> Creates a table of rows and columns.

What is CSS?

CSS (Cascading Style Sheets) is used to style and design the structure created by HTML. It controls layout, colors, fonts, and spacing. This section shows how Bootstrap replaces many manual CSS styles with pre-designed classes.

What is Bootstrap?

Bootstrap is a powerful CSS framework that makes designing websites faster and easier. It includes ready-to-use components such as navigation bars, buttons, tables, forms, and a responsive grid system.

With Bootstrap, developers can quickly create modern and mobile-friendly designs without writing a lot of custom CSS.

Bootstrap Grid System

The grid system helps create responsive layouts. Example: three columns on larger screens that stack on smaller screens.

What is JavaScript?

JavaScript makes web pages interactive. It can validate forms, create animations, and update content dynamically. Bootstrap also provides JavaScript components like modals, carousels, and dropdowns.
